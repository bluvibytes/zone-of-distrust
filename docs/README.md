# Zones of Distrust Documentation

This folder contains the full Zones of Distrust (ZoD) documentation set for the **v0.9 RFC** release.

ZoD is a layered security architecture designed to govern autonomous AI agents through independent enforcement layers that separate reasoning from execution.

For an overview of the project, see the root README: **[/README.md](/README.md)**

# Code of Conduct

We aim to build a constructive, adversarial-friendly security community around Zones of Distrust.

## Expected Behavior
- Be respectful and professional.
- Attack ideas and implementations, not people.
- Provide reproducible reports and evidence when claiming bypasses.
- Credit others’ prior work.

## Unacceptable Behavior
- Harassment, discrimination, threats, or doxxing.
- Sharing private data or secrets.
- Intentionally disrupting project infrastructure.
